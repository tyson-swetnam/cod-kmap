# cod-kmap Bibliography

This page will hold the publications that inform and cite the knowledge
map: source papers for vocabularies and classifications, key reports on
the coastal observing landscape, and studies that use the dataset.

It is an intentional placeholder in this release. Entries will be added
as the research waves close out and the project accumulates a citation
trail.

---

## Planned sections

### Foundational & methodological

Papers and reports that define how coastal observing programs are
catalogued, classified, and funded. Expected entries include:

- National Research Council (2014). *Sustaining Ocean Observations to
  Understand Future Changes in Earth's Climate.*
- IOC-UNESCO. *Global Ocean Observing System — implementation plans.*
- Malone, T. C. et al. *The coastal module of the Global Ocean Observing
  System.*
- Snyder, J. et al. *Maturity matrices for ocean observing data products.*

### Network & program descriptions

Authoritative citations for each network in `schema/vocab/networks.csv`:

- IOOS Regional Associations — program descriptions and evaluation
  reports.
- Ocean Observatories Initiative (OOI) — science plan and annual reports.
- NERRS — System-Wide Monitoring Program (SWMP) protocol documents.
- LTER Network — 50-year review and site briefs.
- Sea Grant — national program biennial reports.

### Controlled vocabulary sources

- NASA GCMD — *Science Keywords* (v. latest) — crosswalked into
  `research_areas.csv`.
- WoRMS — *World Register of Marine Species* (future enrichment).

### Regional coverage

Peer-reviewed inventories of coastal research infrastructure in specific
regions will be added here, e.g.:

- Gulf of Mexico observing syntheses.
- NE Pacific acidification-monitoring networks.
- Caribbean coastal-reef monitoring programs.
- Southern Ocean & South American upwelling observing programs.

### Publications that cite cod-kmap

Once the dataset is public, papers and reports that use it will be
collected here. Contributions welcome via a GitHub pull request that
appends to this file with a standard citation (DOI preferred).

---

## Contributing a citation

Open an issue or PR at
[github.com/tyson-swetnam/cod-kmap](https://github.com/tyson-swetnam/cod-kmap).
Include:

- Full citation (authors, year, title, venue, DOI)
- Which agent(s) or table(s) the reference supports
- A one-sentence note on how the reference is used

All citations are reviewed before being merged.
